About Our Project

This project introduces a worker safety detection tool using image processing technology, integrated with an IoT-based alarm system to monitor and notify any violations in the use of Personal Protective Equipment (PPE) in industrial environments. The tool detects PPE compliance using image processing, and triggers IoT-based alarms to alert supervisors in case of violations. Developed by students from Universitas Andalas, the tool aims to enhance workplace safety by reducing accidents and ensuring adherence to safety standards.
